This web based application shows random quotes on clicking a button. It also has a button to tweet out the quote (the tweet will be auto populated with a #quote ).

API used: Quotes on Design 4 API  - https://quotesondesign.com/api-v4-0/ 

CSS : Bootstrap - https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/css/bootstrap.min.css

Icons : Font Awesome - https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css

JavaScript : jquery - https://cdnjs.cloudflare.com/ajax/libs/jquery/3.2.1/jquery.min.js
